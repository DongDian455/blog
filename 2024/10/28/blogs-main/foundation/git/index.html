git使用

发表于2024-10-28|更新于2024-10-28|计算机基础
|阅读量:

reset

  • hard:会回退到上个版本的已提交状态
  • soft:会回退到上个版本的未提交状态
  • mixed:会回退到上个版本已添加但未提交的状态

    在Git中，用HEAD表示当前版本,上一个版本就是HEAD^, 上上一个版本就是HEAD^^, 当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100。

checkout

该命令的作用是把工作区的修改全部撤销，就是把没有执行add的内容给撤销，回到添加到暂存区后的状态。

栗子(🌰)：
修改readme -> add readme.txt -> 第二次修改readme -> 执行checkout
readme.txt的状态就会回到add readme.txt的状态
