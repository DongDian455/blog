异或的定义

异或的定义为，如何两个值相等则取0，不相等则取1。异或的数学符号使用’⊕’

异或的性质

  1. 任何数和 0 做异或运算，结果仍然是原来的数，即 a⊕0=a 。
  2. 任何数和其自身做异或运算，结果是 0，即 a⊕a=0。
  3. 异或运算满足交换律和结合律，即 a⊕b⊕a=b⊕a⊕a=b⊕(a⊕a)=b⊕0=b。

常见用法

加密

异或运算可以用于加密。
第一步，明文（text）与密钥（key）进行异或运算，可以得到密文（cipherText）。
text ^ key = cipherText

第二步，密文与密钥再次进行异或运算，就可以还原成明文。
cipherText ^ key = text

原理很简单，如果明文是 x，密钥是 y，那么 x 连续与 y 进行两次异或运算，得到自身。

数据备份

异或运算可以用于数据备份。
首先，文件 x 和文件 y 进行异或运算，产生一个备份文件 z。
x ^ y = z
以后，无论是文件 x 或文件 y 损坏，只要不是两个原始文件同时损坏，就能根据另一个文件和备份文件，进行还原。

算法题

给你一个 非空 整数数组 nums ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

思路：
由于相同数字进行异或会等于0，0与任意数异或还是得到原来的数，所以遍历数组进行异或，剩下的数字就是只出现了一次的元素。
